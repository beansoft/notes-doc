::: tip 使用入门
* Step1: 点击 代码笔记 工具窗口 
* Step2: 点击蓝色字体 在用户目录创建数据库
* Step3: 打开项目文件, 编辑并选择要添加笔记的源代码 
* Step4: 右键点击或者按下 Alt+回车 组合键或Alt+M组合键
* Step5: 添加标题, 代码笔记成功保存到了 代码笔记 工具窗口
:::
